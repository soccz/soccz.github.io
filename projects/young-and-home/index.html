Overview

Information Asymmetry Zero

부동산 지식이 없는 사회초년생도, AI와 함께라면 전문가처럼. 단순한 매물 검색을 넘어, 계약 안전 진단부터 집주인과의 협상, 숨겨진 금융 혜택까지 전 과정을 5명의 AI 에이전트가 코디네이팅합니다.

실질 비용 계산
"월세 50만원" 집이 내 조건(연소득, 자산, 나이)에서 정부 지원 대출과 보조금을 적용하면 "실질 월세 12만원"이 되는 마법을 보여줍니다.
계약 안전 진단
등기부등본 PDF를 업로드하면 소유자 일치 여부, 근저당 비율, 경매 위험도를 AI가 자동 분석합니다. 독소 조항도 탐지합니다.
24시간 감시
n8n + Notion 연동으로 관심 지역의 공고를 24시간 모니터링합니다. Slack 알림과 Notion 대시보드에 실시간 적재됩니다.
Multi-Agent System

5명의 AI 에이전트

하나의 문제를 5명의 전문가가 동시에 바라봅니다. 단순한 LLM 래퍼가 아니라, 각기 다른 전문성을 가진 에이전트들이 LangGraph 위에서 협업하는 Multi-Agent System입니다.

Agent 1 — Recommender
매물 추천 에이전트
사용자의 예산, 통근 거리, 선호 조건을 분석하여 최적의 매물을 추천합니다. RAG 기반 검색으로 정확한 매칭을 수행합니다.
RAG벡터 검색조건 매칭
Agent 2 — Safety Analyzer
안전 진단 에이전트
등기부등본, 계약서를 분석하여 근저당 비율, 소유자 일치 여부, 경매 위험도를 자동 진단합니다. 특약 사항의 독소 조항도 탐지합니다.
PDF 파싱위험도 스코어링독소 조항 탐지
Agent 3 — Negotiator
협상 보조 에이전트
시세 데이터와 매물 조건을 기반으로 협상 전략을 제안합니다. 집주인과의 커뮤니케이션 템플릿도 생성합니다.
시세 분석협상 전략템플릿 생성
Agent 4 — Financial Agent
금융 계산 에이전트
사용자의 연소득, 자산, 나이를 분석하여 적용 가능한 정부 지원 대출(중기청, 청년버팀목)과 보조금을 자동 계산합니다.
대출 시뮬레이션전세 vs 월세실질 비용
Agent 5 — Monitor
실시간 감시 에이전트
n8n 워크플로우와 연동하여 관심 지역의 매물 공고와 뉴스를 24시간 감시합니다. SHA256 해시 기반 Smart Filter로 API 비용을 90% 절감합니다.

Smart Hash Filter

들어오는 모든 데이터의 SHA256 해시를 생성하여 DB와 대조합니다(0.01초). 변화가 없으면 API 연산을 건너뛰고 DB에 저장된 과거의 Risk Score를 리턴합니다. 이를 통해 API 비용을 90% 절감하면서도 사용자에게는 언제나 유효한 점수를 보여줍니다.

SQLite Caching

가벼운 SQLite를 활용해 RAG 검색 결과와 금융 시뮬레이션 결과를 캐싱하여 응답 속도를 3초에서 0.1초로 단축했습니다.

Key Features

주요 기능

복잡한 부동산 절차를 AI가 단순화합니다.

안전 진단 (Safety Scan)
등기부등본 PDF를 업로드하면 소유자 일치 여부, 근저당 비율, 경매 위험도를 분석합니다. 강제퇴거 조항 같은 독소 조항도 자동 탐지합니다.
24시간 등기 모니터링
n8n + Notion 연동으로 관심 지역의 공고를 24시간 감시합니다. 분석 결과는 Notion 데이터베이스에 실시간 적재되고, Markdown 리포트로 저장됩니다.
금융 계산기 (Smart Calculator)
전세 vs 월세 비교 시 대출 이자와 관리비까지 포함하여 2년 거주 총비용을 정확하게 계산합니다. 정부 지원 프로그램 자동 매칭.
