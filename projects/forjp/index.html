Overview

프로젝트 소개

COUPLE은 대중교통 기반 데이트 코스 플래너입니다. Kakao Local로 장소를 검색하고, ODsay로 이동 시간을 계산하며, 시간대별 혼잡도와 대기 시간을 시뮬레이션합니다.

코스 자동 생성
출발지와 선호 카테고리를 기반으로 식사, 카페, 활동, 산책 등의 장소를 조합한 데이트 코스를 자동으로 추천합니다.
대중교통 최적화
ODsay API를 통해 장소 간 대중교통 이동 시간을 실시간으로 계산하고, 이동이 효율적인 코스를 우선 추천합니다.
저장 & 공유
마음에 드는 코스를 Supabase에 저장하고, 링크를 통해 상대방과 공유할 수 있습니다. 로컬 저장 fallback도 지원합니다.

Architecture

시스템 구조

Provider 패턴으로 데이터 소스를 추상화하여, API 키 유무에 따라 live/mock이 자동 전환됩니다.

Provider 계층
각 데이터 소스는 live, hybrid, mock 세 가지 모드로 동작합니다. 환경 변수가 없으면 자동으로 mock fallback합니다.
live
전부 실데이터
hybrid
일부 실데이터 + fallback
mock
전부 목업 fallback
API 엔드포인트
추천, 커스텀 플랜, 플래너, 저장, 진단까지 6개 API 엔드포인트로 구성됩니다.

API Routes
3
Supabase 테이블

Providers

데이터 소스

장소 검색, 대중교통, 리뷰, 저장, 캐시 각각에 독립적인 provider가 할당됩니다.

Kakao Local
장소 검색 실데이터 provider. 카테고리 매칭, 거리 필터, 중복 제거 등 품질 필터가 적용됩니다.
ODsay Transit
대중교통 이동 시간 계산 실데이터 provider. 장소 간 최적 경로와 소요 시간을 산출합니다.
Supabase
코스 저장, 추천 캐시, 리뷰 요약 저장소. saved_plans, api_cache, place_review_summaries 테이블로 구성됩니다.
