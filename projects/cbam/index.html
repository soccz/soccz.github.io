Background

What is CBAM?

EU 탄소국경조정메커니즘(CBAM)은 철강, 시멘트, 알루미늄, 비료, 수소, 전력 등 EU로 수입되는 제품에 부과되는 탄소 관세입니다.

CBAM 작동 원리

EU 수입자는 EU ETS 경매 가격에 연동된 CBAM 인증서를 구매해야 합니다. 비용은 EU와 수출국 간 탄소 가격 차이를 반영하며, 한국 기업은 EU 바이어의 가격 재조정을 통해 간접적으로 비용을 부담하게 됩니다.

CBAM Cost = Export Volume × [Emission Factor − Benchmark × (1 − Phase-in Rate)] × EUA Price × FX Rate
Source: EU Regulation 2023/956, Art. 7

System Design

Architecture

CarbonCast는 ESG TradeGuard 플랫폼 내에서 3개 통합 레이어로 운영됩니다.

Layer 1 — AI Engine
Emission Estimation + Industry Impact + Compliance
분포 무관 배출계수 추정을 위한 Conformal Prediction, 산업 전반 CBAM 비용 전파 분석을 위한 Leontief I-O 모델 + GNN, 자동 규제 가이드를 위한 LLM 기반 RAG 에이전트.
LightGBM / XGBoost MAPIE Conformal Isolation Forest PyTorch Geometric GNN LLM + RAG
Layer 2 — Trust Layer
Blockchain Audit Trail + Digital Product Passport
검증된 배출 데이터의 SHA-256 해시 기반 위변조 방지 기록. EU 2027 의무화에 앞서 공급망 탄소 추적을 위한 Digital Product Passport(DPP). 프라이버시 보호 크로스 컴퍼니 모델 학습을 위한 Federated Learning.
Ethereum Sepolia Hyperledger Fabric Web Crypto API Federated Learning
Layer 3 — Financial Layer
Trade Finance Integration + Portfolio Analysis
수출 서류 자동 HS코드 추출을 위한 READIT AI-OCR. PCAF 기준 포트폴리오 탄소 익스포저 분석. CBAM 비용이 EBITDA 및 이자보상비율에 미치는 영향을 반영한 신용평가 연동.
READIT OCR PCAF Framework DART Open API NGFS Scenarios
Core Modules

Key Components

3개 운영 모듈이 배출 데이터 입력부터 검증된 비용 시뮬레이션까지 처리합니다.

01
AI Quality Scoring
EU 공인 검증기관 심사 전 배출 데이터 신뢰도를 사전 검증합니다. 4개 기준에 대한 상세 체크 결과와 함께 0-100 품질 스코어를 출력합니다.
Scoring Criteria
배출계수 범위(50pt), 에너지 범위(20pt), 에너지-배출 일관성(20pt), 생산 규모(5pt)
Thresholds
80+: 신뢰 / 60-79: 검토 권고 / 60 미만: 재확인 필요
Training Data
K-ETS 레지스트리 (~800 시설)
Method
Isolation Forest + KDE 다변량 이상탐지
02
Blockchain Verification
품질 검증된 배출 데이터의 불변 감사 추적을 생성합니다. 60점 이상 데이터만 블록체인에 기록됩니다. EU CBAM 레지스트리와 별도인 자체 감사 인프라입니다.
On-chain
SHA-256 해시, 블록 번호, 타임스탬프, 익명화 기업 ID, 품질 스코어
Off-chain
원본 전체 데이터 (하나금융 DB)
Prototype
Ethereum Sepolia 테스트넷
Production
Hyperledger Fabric (허가형)
03
CBAM Cost Simulation
4개 NGFS 기후 시나리오 하에서 2026-2034년 CBAM 비용 경로를 산출합니다. EU 기본값 대비 실측 데이터의 절감 잠재력을 정량화합니다.
Scenarios
Net Zero 2050, Below 2C, Delayed Transition, Current Policies

Pipeline

Data Flow

두 가지 트랙이 서로 다른 유형의 기업에 대응합니다: 데이터 보유 기업과 배출 데이터 미보유 중소기업.

Track A: 배출 데이터 보유 기업

1

서류 업로드

READIT OCR로 수출 서류 처리. HS코드, 수량, 목적지 자동 추출.

2

배출 데이터 입력

기업이 배출계수, 공정 유형, 에너지 소비 데이터를 제공.

3

AI Quality Scoring

4개 기준 분석과 함께 0-100 품질 스코어. EU 검증 전 사전 검증.

4

블록체인 기록

60점 이상: SHA-256 해시 온체인 기록. 위변조 방지 감사 추적 생성.

5

비용 시뮬레이션

실측 배출 데이터 기반 4개 NGFS 시나리오 하 2026-2034 CBAM 비용 경로.

Track B: 배출 데이터 미보유 기업

1

기본 정보만 입력

산업 섹터와 수출 물량. 배출 데이터 불필요.

2

기본값 시뮬레이션

EU 기본값 + 가산 페널티로 CBAM 비용 산출.

3

절감 잠재력

"실측 데이터를 확보하면 연간 X억원을 절감할 수 있습니다."

4

측정 동기 부여

정량화된 절감액이 기업의 배출 데이터 인프라 투자 유인을 생성.

Project Context

Hana Financial Group
