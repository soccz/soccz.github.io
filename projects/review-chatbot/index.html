Overview

프로젝트 개요

직접 수집한 리뷰 데이터(end.csv)를 기반으로 사용자의 취향에 맞는 음식점을 추천하고, 왜 그 음식점이 적합한지 이유를 함께 제공하는 챗봇 시스템입니다.

리뷰 데이터 수집
웹 크롤링을 통해 음식점 리뷰 데이터를 직접 수집하고 CSV 형태로 정제합니다. 리뷰 텍스트, 평점, 음식점 정보를 포함합니다.
텍스트 분석
수집된 리뷰 텍스트를 NLP 기법으로 분석하여 각 음식점의 특성과 장단점을 파악합니다.
대화형 추천
챗봇 인터페이스를 통해 사용자의 요구사항을 파악하고, 리뷰 분석 결과를 바탕으로 음식점을 추천하며 추천 이유를 설명합니다.

챗봇 대화 예시

사용자가 원하는 조건을 입력하면, 리뷰 데이터를 기반으로 적합한 음식점을 추천하고 그 이유를 설명합니다.

USER
분위기 좋은 이탈리안 레스토랑 추천해줘
CHATBOT
리뷰 분석 결과, "트라토리아 서울"을 추천드립니다. 총 47건의 리뷰 중 "분위기"를 언급한 리뷰가 23건이며, 그 중 91%가 긍정적입니다. 특히 "조용하고 고급스러운 분위기"라는 평이 반복적으로 나타납니다.
USER
가격대는 어때?
CHATBOT
리뷰에서 "가성비"를 언급한 15건 중 60%가 "가격 대비 만족스럽다"는 평가입니다. 다만 평균 객단가가 높은 편이라는 의견도 있으니 참고해주세요.
