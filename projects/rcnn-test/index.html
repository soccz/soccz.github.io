Overview

프로젝트 개요

Matterport의 Mask R-CNN 구현체를 기반으로 패션 이미지에서 의류 아이템을 픽셀 단위로 분할하는 인스턴스 세그멘테이션 실험입니다. COCO 사전학습 가중치에서 출발하여 46개 패션 카테고리로 전이학습합니다.

Instance Segmentation
단순 객체 탐지(Bounding Box)를 넘어 각 객체의 픽셀 수준 마스크를 예측합니다. 하나의 이미지 안에 복수의 의류 아이템이 겹쳐 있어도 개별 인스턴스를 구분합니다.
Transfer Learning
COCO 데이터셋으로 사전학습된 가중치를 로드한 뒤, 분류/회귀 헤드를 패션 도메인에 맞게 재학습합니다. 헤드만 먼저 학습 후 전체 레이어를 미세조정하는 2단계 전략을 사용합니다.
Multi-label Classification
하나의 세그먼트가 복수의 속성을 가질 수 있는 멀티라벨 구조입니다. ClassId에서 기본 카테고리를 분리하여 46개 클래스 체계로 정리합니다.

Architecture

모델 구조

Mask R-CNN은 Faster R-CNN에 마스크 예측 브랜치를 추가한 구조입니다. 각 구성 요소의 역할과 설정을 정리합니다.

Backbone
ResNet-50 + FPN
Feature Pyramid Network를 통해 다중 스케일 특성맵을 생성하여, 크기가 다양한 패션 아이템을 효과적으로 탐지합니다.
ResNet50FPNMulti-scale
Region Proposal
RPN (Region Proposal Network)
특성맵 위에서 앵커 박스를 기반으로 객체가 존재할 가능성이 높은 영역을 제안합니다. 제안된 영역은 RoI Align을 통해 고정 크기로 변환됩니다.
Anchor BoxRoI AlignNMS
Head
Classification + Mask Branch
각 RoI에 대해 클래스 분류, 바운딩 박스 회귀, 픽셀 마스크 예측을 병렬로 수행합니다. COCO 헤드를 제거하고 46+1 클래스로 재구성합니다.

Training

학습 전략

COCO 사전학습 가중치에서 출발하여 단계적으로 학습 범위를 확장하는 전략을 사용합니다.

Stage 1 — Heads Only
백본을 동결하고 분류/마스크 헤드만 학습합니다.
Learning Rate: 2×10-4
Epochs: 2
증강: 미적용
Stage 2 — All Layers
전체 레이어를 미세조정합니다.
Learning Rate: 1×10-4
Epochs: 6
증강: 좌우 반전 (Fliplr 0.5)
Stage 3 — Fine-tuning
낮은 학습률로 추가 미세조정합니다.
Learning Rate: 5×10-5
Epochs: 8
증강: 좌우 반전 (Fliplr 0.5)

Notes

코드 리뷰 노트

Kaggle CNN 관련 코드를 리뷰하며 정리한 참고 사항입니다. 코드에 일부 오류가 있으나 전체적인 파이프라인 이해를 위해 기록합니다.

COCO 가중치 전이학습 전략 Transfer Learning
mrcnn_class_logits, mrcnn_bbox_fc, mrcnn_bbox, mrcnn_mask 레이어를 제외하고 가중치를 로드합니다. 도메인 특화 헤드만 새로 학습하는 표준적인 접근입니다.
RLE 인코딩/디코딩 구현 Data Processing
Run-Length Encoding으로 압축된 마스크 데이터를 2D 배열로 복원하고, 추론 결과를 다시 RLE로 인코딩하여 제출 형식에 맞춥니다.
멀티라벨 세그먼트 처리 Preprocessing
ClassId에 언더스코어(_)가 포함된 멀티라벨 데이터를 기본 카테고리와 속성으로 분리합니다. 기본 카테고리만 추출하여 46개 클래스 체계를 구성합니다.
데이터 증강의 최소 적용 Augmentation
imgaug의 Fliplr(0.5)만 적용하는 보수적인 증강 전략입니다. 패션 이미지의 좌우 대칭 특성을 활용하되 과도한 변형은 피합니다.
추론 시 테스트 시간 증강 (TTA) Inference
추론 단계에서 원본과 좌우 반전 이미지를 모두 예측한 뒤 결과를 병합하여 검출 성능을 향상시킵니다.
