Overview

프로젝트 개요

나스닥의 섹터별 평균 수익률과 업비트에 상장된 원화 코인들 사이의 통계적 관계를 분석합니다. 주식 시장의 추세를 따라가는 암호화폐가 존재하는지 검증하고, 3일 시차의 변수 관계를 파악합니다.

상관관계 분석
나스닥 섹터별 수익률과 업비트 코인 가격 간의 피어슨/스피어만 상관계수를 산출하여 크로스마켓 연관성을 측정합니다.
PCA 차원 축소
주성분 분석을 통해 고차원 변수 공간의 핵심 구조를 추출하고, 주식 추세를 추종하는 코인 클러스터를 식별합니다.
시차 변수 분석
나스닥의 3일 전 데이터와 코인 데이터 간의 리드-래그 관계를 파악하여 예측 가능한 패턴을 탐색합니다.
Methodology

분석 방법론

크로스마켓 상관관계와 차원 축소를 결합한 다단계 분석 프레임워크입니다.

Pipeline

분석 파이프라인

변수 추출 폴더에 있는 코드를 통해 나스닥과 코인 데이터의 관계를 분석하고, 결과를 시각화하여 저장합니다.

변수 추출
나스닥 섹터 데이터와 암호화폐 데이터 간의 핵심 변수를 추출하고 시차를 적용하여 리드-래그 관계를 정량화합니다.
시각화 저장
분석 결과를 이미지 파일로 저장하는 코드가 포함되어 있어 반복적인 분석과 결과 비교가 용이합니다.
예측 모델
Autoformer 등 논문 기반의 시계열 예측 모델을 코드로 구현하여 추가적인 예측 파이프라인을 구축할 계획입니다.
