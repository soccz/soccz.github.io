Overview

운동을 자산으로 관리하다

Iron Quant는 운동 기록을 금융 투자에 비유합니다. 몸의 근력은 자산 포트폴리오, 각 운동은 매매, 앱은 개인 퀀트 애널리스트입니다. 스쿼트 100 5 5 한 줄이면 볼륨, 추정 시간, 칼로리, 1RM까지 자동 계산됩니다.

Text Parsing

자연어 운동 입력

벤치 60 10 5 @9 형태로 입력하면 종목, 중량, 횟수, 세트, RPE를 자동 파싱합니다. 웹과 텔레그램 모두 동일한 파서를 사용하며, 미리보기로 파싱 결과를 바로 확인할 수 있습니다.

1RM Estimation

Epley 공식 기반 1RM

Big 3(스쿼트, 벤치, 데드리프트)의 1RM을 Epley 공식으로 추정합니다. 12회 이상 고반복 세트의 과대 추정을 방지하기 위해 rep clamp를 적용하며, PR은 최대값만 유지됩니다.

Portfolio Analysis

투자 의견 자동 생성

최근 5회 운동을 분석해 상/하체 볼륨 불균형, 비활동 경고(4일+), 과훈련 감지(평균 RPE > 8.5)를 수행합니다. 결과는 Buy/Hold/Sell 형태의 투자 의견으로 전달됩니다.

Quant Engine

퀀트 분석 시스템

총 자산 가치(Total Asset Value)는 3대 운동 1RM의 가중합으로 계산됩니다. 벤치프레스에는 1.2x 가중치가 적용되며, 이 값에 10을 곱해 직관적인 "자산" 수치를 만듭니다.

// 총 자산 가치 계산
const totalAsset = (
  user.estimated_1rm_squat  * 1.0 +
  user.estimated_1rm_bench  * 1.2 +
  user.estimated_1rm_dead   * 1.0
) * 10;

// Epley 1RM 추정 (rep clamp 적용)
function estimate1RM(weight, reps) {
  const clampedReps = Math.min(reps, 12);
  return weight * (1 + clampedReps / 30);
}

Telegram Bot

Engineering Details

설계 결정과 구현 디테일

단일 사용자 앱이지만 보안, 오프라인 대응, 데이터 무결성에서 의도적인 설계 결정이 있습니다.

Security

HMAC-SHA256 앱 잠금

로그인 시스템 대신 공유 시크릿 기반 쿠키 인증을 사용합니다. crypto.timingSafeEqual로 타이밍 공격을 방지하며, 헤더(x-app-secret)와 쿠키 두 경로를 모두 지원합니다.

Data Integrity

PR은 최대값만 유지

1RM PR은 append-only로 단일 운동이 기존 값을 낮출 수 없습니다. 전체 히스토리 기반 재계산이 필요하면 /recompute 명령어로 명시적으로 실행합니다.

Rate Limiting

인메모리 토큰 버킷

Telegram 웹훅은 채팅당 30 req/min, cron 엔드포인트는 IP당 60 req/min으로 제한합니다. 서버리스 환경에서도 cold start 간 리셋되는 수준의 방어입니다.

Observability

Request ID 트레이싱

모든 API 응답에 x-request-id 헤더를 포함합니다. 텔레그램 봇의 /debug 명령어로 DB 연결, 웹훅 상태, 환경변수를 한번에 진단할 수 있습니다.

Cron Dedup

리마인더 중복 방지

GitHub Actions가 10분마다 실행되지만, telegram_last_reminded_date 컬럼과 사용자 timezone 기반 10분 윈도우 체크로 하루에 한 번만 알림을 전송합니다.

Calorie Estimation

MET 기반 칼로리 계산

사용자 체중, 추정 운동 시간, RPE를 조합한 MET(Metabolic Equivalent of Task) 기반 칼로리 추정입니다. 별도 센서 없이 입력 데이터만으로 근사합니다.

Design Philosophy

전체 UX 언어가 금융 투자 메타포로 통일되어 있습니다. 상/하체 볼륨 분석은 "섹터 분석", 장기간 미운동은 "시장 폭락 경고", 과훈련은 "변동성 체크"로 표현합니다. Toss 스타일 디자인 시스템(커스텀 컬러 토큰, 다크 모드)을 적용했으며 PWA로도 동작합니다.
