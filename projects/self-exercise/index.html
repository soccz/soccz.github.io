Overview

프로젝트 개요

Iron Quant는 감량 모드와 근육 모드를 지원하는 단일 사용자 운동 기록 앱입니다. 웹 앱에서 상세 분석을, 텔레그램에서 빠른 기록을 남길 수 있습니다. 목표에 맞는 추천과 자동 분석이 핵심입니다.

목표 기반 추천
감량 모드에서는 유산소 시간과 회복 중심으로, 근육 모드에서는 중량과 볼륨 중심으로 오늘의 운동을 추천합니다.
듀얼 입력 채널
웹 앱에서 여러 줄 텍스트 입력 또는 텔레그램 고정 버튼으로 빠르게 기록. 동기화 상태를 실시간으로 확인할 수 있습니다.
자동 분석
최근 기록, 주간 요약, 1RM 추적 등을 자동으로 계산합니다. CSV/JSON 내보내기도 지원합니다.
Modes

감량 vs 근육

두 가지 모드에 따라 홈 화면의 핵심 지표, 추천 방향, 기록 템플릿이 달라집니다.

Fat Loss Mode
감량 모드
  • 홈: 주간 유산소 목표 진행률, 칼로리
  • 추천: 유산소 시간 / 회복 중심
  • 입력 예시: 러닝머신 30분 8km/h 경사 1
  • 입력 예시: 사이클 35분 20km/h
Muscle Gain Mode
근육 모드
  • 홈: 3대 1RM, 자산 지표
  • 추천: 중량 / 볼륨 중심
  • 입력 예시: 스쿼트 100 5 5
  • 입력 예시: 벤치 60x10x5 @9

Telegram

텔레그램 봇 명령어

고정 버튼 6개와 슬래시 명령어로 빠르게 기록하고 조회합니다. cron 브리핑 메시지에서도 동일한 버튼이 유지됩니다.

명령어설명
/status현재 상태 리포트
/rec오늘 추천
/mode fat|muscle모드 전환
/cond sleep 7 fatigue 4 ...오늘 컨디션 입력
/last마지막 운동 조회
/edit ...마지막 운동 수정
/undo마지막 운동 삭제
/export csv|json데이터 내보내기
/remind ...리마인더 설정
/debug연결 상태 점검
