# 통계 도구상자: 논문의 숫자를 읽는 법

> 이 논문에는 7개의 테이블과 6개의 그림이 있다. 테이블에는 수백 개의 숫자가 빼곡하다. 이 챕터는 **그 숫자 하나하나가 뭘 의미하는지** 확실히 잡아주는 도구상자다. 여기를 건너뛰면 논문 본론이 암호처럼 보일 것이다.

---

## 평균(Mean): "이 전략이 평균적으로 얼마 버는가"

가장 기본. 매달 수익률을 기록하고, 전부 더해서 개수로 나눈다.

$$\bar{r} = \frac{1}{T}\sum_{t=1}^{T} r_t$$

이 논문에서 "Mean = 3.9%"라 하면, **연환산** 평균 초과수익률이 3.9%라는 뜻이다.
(월평균 × 12로 연환산)

### 주의: 이건 "확실히 3.9% 번다"가 아니다

평균은 과거의 결과일 뿐이다. 어떤 달은 +8%, 어떤 달은 -5% — 이 출렁임을 무시하면 안 된다. 그래서 다음 개념이 필요하다.

---

## 표준편차(Standard Deviation, Stdev): "얼마나 출렁이는가"

평균 주위로 수익률이 얼마나 흩어져 있는가.

$$\sigma = \sqrt{\frac{1}{T-1}\sum_{t=1}^{T}(r_t - \bar{r})^2}$$

쉽게: "매달 수익률이 평균에서 얼마나 벗어나는가"의 대표값.

### 일상 비유

- 키 평균 170cm, 표준편차 5cm → 대부분 165~175cm 사이
- 키 평균 170cm, 표준편차 20cm → 150~190cm까지 들쑥날쑥

투자에서:
- "Stdev = 14.8%" → 수익률이 연간 약 ±14.8% 범위에서 움직인다
- 평균이 3.9%이고 표준편차가 14.8%면: 좋은 해에 +18.7%, 나쁜 해에 -10.9% 정도

### 연환산

이 논문은 월간 표준편차에 $\sqrt{12}$를 곱해 연환산한다.

$$\sigma_{annual} = \sigma_{monthly} \times \sqrt{12}$$

왜 12가 아니라 $\sqrt{12}$인가? 수익률이 독립이라면, 분산(표준편차의 제곱)은 시간에 비례하고, 표준편차는 시간의 제곱근에 비례하기 때문이다.

## 샤프 비율(Sharpe Ratio): "위험 대비 수익의 효율"

**이 논문에서 가장 중요한 숫자 중 하나.**

두 전략이 모두 연 10% 수익을 낸다. 하나는 표준편차 5%, 다른 하나는 표준편차 30%. 어느 쪽이 더 좋은 전략인가? 당연히 전자다 — 같은 수익을 더 적은 출렁임으로 달성했으니까.

$$\text{Sharpe Ratio} = \frac{\bar{r} - r_f}{\sigma} = \frac{\text{평균 초과수익률}}{\text{표준편차}}$$

이 논문에서 수익률이 이미 초과수익률(excess return)이므로:

$$\text{Sharpe} = \frac{\bar{r}}{\sigma}$$

### 샤프 비율 벤치마크

| 샤프 비율 | 해석 | 비교 대상 |
|---------|------|---------|
| 0.2~0.3 | 평범 | 미국 주식시장 장기 평균 |
| 0.4~0.5 | 괜찮음 | 잘 분산된 포트폴리오 |
| 0.7~0.9 | 좋음 | 이 논문의 개별 전략 수준 |
| 1.0 이상 | 매우 좋음 | 헤지펀드 목표 수준 |
| **1.5 이상** | **놀라움** | **이 논문의 글로벌 50/50 조합** |

> Table I에서 "Sharpe = 0.26"이면 평범, "Sharpe = 1.59"이면 놀라운 수준이라고 즉시 판단할 수 있다.

## t-통계량(t-statistic): "이게 진짜인가, 운인가?"

### 왜 필요한가

평균 수익률이 3.9%라 해도, 이게 **진짜로 돈을 버는 전략**인지, **우연히 운이 좋았던 것**인지 구분해야 한다.

동전을 10번 던져서 7번 앞면이 나왔다. 이 동전이 편향된 건가? 아니면 운이 좋은 건가?

t-통계량은 이 질문에 답한다: **"이 결과가 순전히 우연일 확률이 얼마나 되는가?"**

### 계산

$$t = \frac{\bar{r}}{\sigma / \sqrt{T}} = \frac{\text{평균}}{\text{표준오차}}$$

- 분자: 평균 수익률 (클수록 좋음)
- 분모: 표준오차 = 표준편차 ÷ $\sqrt{\text{관측치 수}}$ (관측치가 많을수록 작아짐)

**t-통계량은 "신호 대 잡음 비율(signal-to-noise ratio)"이다.** 평균 수익(신호)이 크고, 출렁임(잡음)이 적고, 데이터가 길수록(T가 클수록), t가 커진다.

### 해석 기준

| |t| | 의미 | p-value |
|-----|------|---------|
| < 1.0 | 우연일 가능성 높음 | > 30% |
| 1.0~1.64 | 약한 증거 | 10~30% |
| **1.65~1.96** | **경계선** | **5~10%** |
| **≥ 1.96** | **유의하다 (5% 수준)** | **< 5%** |
| ≥ 2.58 | 매우 유의하다 (1% 수준) | < 1% |
| ≥ 3.29 | 극도로 유의 (0.1% 수준) | < 0.1% |

### 이 논문에서의 사용

Table I의 모든 행에서 평균 아래 괄호 안 숫자가 t-통계량이다:

```
Mean     3.9%
(t-stat) (1.66)    ← t = 1.66, 10% 수준에서 유의하지만 5%는 아님
```

```
Mean     5.8%
(t-stat) (2.49)    ← t = 2.49, 5% 수준에서 유의! 우연일 확률 < 5%
```

> **실전 규칙**: 이 논문의 테이블을 볼 때, 괄호 안 숫자(t-stat)가 2 이상이면 "이건 진짜다"로, 1~2면 "증거가 약하다"로, 1 미만이면 "무시해도 된다"로 빠르게 판단하면 된다.

## 상관계수(Correlation): "두 변수가 같이 움직이는 정도"

### 범위: -1 ~ +1

| 상관 | 의미 | 예시 |
|------|------|------|
| +1.0 | 완벽히 같이 움직임 | A가 오르면 B도 항상 오름 |
| +0.5 | 상당히 같이 | A가 오르면 B도 보통 오름 |
| 0 | 무관 | A의 움직임으로 B를 예측 불가 |
| -0.5 | 상당히 반대로 | A가 오르면 B는 보통 내림 |
| -1.0 | 완벽히 반대로 | A가 오르면 B는 항상 내림 |

### 이 논문에서 핵심적인 상관계수들

1. **가치 간 상관**: 미국 가치와 영국 가치 = +0.68 → 같이 움직인다
2. **모멘텀 간 상관**: 미국 모멘텀과 일본 모멘텀 = +0.65 → 같이 움직인다
3. **가치-모멘텀 상관**: 가치와 모멘텀 = 약 -0.5 ~ -0.6 → 반대로 움직인다

Table II는 상관계수만으로 이루어진 **상관 매트릭스**다. 읽는 법:
- 대각선: 자기 자신과의 상관(= 같은 전략끼리 다른 시장에서의 평균 상관)
- 위/아래: 서로 다른 전략 간 상관

### 분기별(quarterly) 상관을 쓰는 이유

Table II 주석: 상관계수는 **분기별 수익률**로 계산한다. 왜?

서로 다른 시장은 거래 시간이 다르다 (미국 주식시장과 일본 원자재시장은 시차가 있다). 월간 수익률로 상관을 계산하면, 이 **비동시 거래(nonsynchronous trading)** 때문에 상관이 과소추정될 수 있다. 분기별로 하면 이 문제가 완화된다.

## 알파(Alpha)와 베타(Beta): 회귀분석의 핵심

### 왜 필요한가

전략이 연 10% 수익을 냈다. 하지만 같은 기간 주식시장 전체도 10% 올랐다면? 이 전략은 그냥 시장을 따라간 것일 뿐, 특별한 능력이 없다.

**알파**: 시장 움직임을 제거한 뒤 남는 **순수 초과수익**
**베타**: 시장에 대한 **노출도** (시장이 1% 움직일 때 전략이 몇 % 움직이는가)

### 회귀분석

$$r_t = \alpha + \beta \times r_{m,t} + \varepsilon_t$$

- $r_t$: 전략의 초과수익률 (시점 $t$)
- $r_{m,t}$: 시장의 초과수익률
- $\alpha$: **알파** — 시장과 무관하게 번 돈
- $\beta$: **베타** — 시장 노출도
- $\varepsilon_t$: 설명 안 되는 나머지 (잔차)

### 알파의 의미

| 알파 | 의미 |
|------|------|
| 양수, 유의 | **"이 전략은 시장과 무관하게 돈을 벌었다"** — 가치 있는 발견 |
| 0에 가까움 | 시장 위험으로 설명 가능 — 특별할 것 없음 |
| 음수 | 시장 대비 손해 — 나쁜 전략 |

Table I에서:
```
Alpha   5.8%
(t-stat) (2.49)
```
→ "이 전략은 시장 위험을 보정한 후에도 연 5.8%의 초과수익을 냈고, 이것이 우연일 확률은 5% 미만이다."

## $R^2$: "모형이 얼마나 잘 설명하는가"

$$R^2 = 1 - \frac{\text{설명 안 된 변동}}{\text{전체 변동}}$$

- $R^2 = 0$: 모형이 아무것도 설명 못 함
- $R^2 = 0.5$: 변동의 50%를 설명
- $R^2 = 1$: 완벽하게 설명

### 이 논문에서의 사용

**시계열 $R^2$**: 개별 포트폴리오의 수익률 변동 중 팩터로 설명되는 비율
- Table VI: 글로벌 3팩터 모형의 평균 시계열 $R^2$ = 0.59

**횡단면 $R^2$**: 48개 포트폴리오의 **평균 수익률 차이**를 모형이 얼마나 설명하는가
- Table VI: 글로벌 3팩터의 횡단면 $R^2$ = 0.72

> 시계열 $R^2$와 횡단면 $R^2$는 다른 것이다. 시계열은 "시간에 따른 출렁임", 횡단면은 "포트폴리오 간 수익률 차이"를 설명하는 비율.

## 주성분분석(PCA): 공통 움직임을 찾다

### 왜 필요한가

8개 시장에서 가치 8개, 모멘텀 8개 = 총 16개 전략이 있다. 이 16개가 각각 독립적으로 움직이는가, 아니면 **공통의 요인**에 의해 같이 움직이는가?

### 일상 비유

교실에 30명의 학생이 있다. 모든 학생의 성적(수학, 영어, 과학...)을 분석해보니, "IQ"라는 하나의 숫자로 성적 변동의 60%를 설명할 수 있었다. 이 "IQ"가 **첫 번째 주성분(first principal component, PC1)**이다.

### 기술적 설명

16개 전략의 수익률로 **공분산 행렬(covariance matrix)**을 만든다. 이 행렬의 **고유벡터(eigenvector)**가 주성분이고, **고유값(eigenvalue)**이 각 주성분의 설명력이다.

첫 번째 주성분 = 전체 변동 중 가장 큰 부분을 설명하는 단일 방향

### Figure 1이 보여주는 것

이 논문의 Figure 1:
- **상단**: 4개국 개별주식의 가치/모멘텀 → PC1이 **54%** 설명
  - 모든 가치 전략에 양(+)의 가중치, 모든 모멘텀 전략에 음(-) 가중치
  - 즉, PC1 = "**롱 모멘텀 + 숏 가치**" (또는 그 반대)
- **하단**: 전 자산군 → PC1이 **23%** 설명
  - 같은 패턴: 가치와 모멘텀이 반대 방향으로 로드

**해석**: 가치와 모멘텀의 가장 큰 공통 움직임은 "가치가 오를 때 모멘텀이 내리고, 모멘텀이 오를 때 가치가 내리는" 패턴이다. 이게 **전 세계적으로** 나타난다.

## Fama-MacBeth 회귀: 횡단면 가격결정

### 왜 보통 회귀와 다른가?

보통 회귀: 한 자산의 수익률을 시간에 따라 분석 (시계열)
Fama-MacBeth: **매 시점마다** 여러 자산의 수익률을 설명 (횡단면), 그 계수의 시계열 평균

### 절차

1. 매 월 $t$마다: 48개 포트폴리오의 수익률을 베타(위험 노출도)에 대해 횡단면 회귀
   $$r_{i,t} = \lambda_{0,t} + \lambda_{1,t} \beta_{i,t}^{liquidity} + \lambda_{2,t} \beta_{i,t}^{value} + ... + \varepsilon_{i,t}$$

2. 매 월의 계수 $\lambda_{1,t}, \lambda_{2,t}, ...$을 모은다

3. 이 계수들의 **시계열 평균**과 **t-통계량**을 계산

### 왜 이렇게 복잡하게?

패널 데이터(시간 × 자산)에서 보통 회귀를 하면, 같은 시점의 잔차가 **상관**되어 있어서 t-통계량이 과대평가된다. Fama-MacBeth 방법은 이 문제를 해결한다.

### Table V 읽는 법

```
β_Liquidity_risk   0.0024
                    (3.05)
```
→ "유동성 리스크 베타가 1단위 높아지면, 월 수익률이 0.24bp(=약 3%/년) 높다. t-통계량 3.05로 매우 유의."

이 계수는 동시에 **팩터 모방 포트폴리오(factor mimicking portfolio)**의 수익률이기도 하다. 즉, 유동성 리스크의 가격(risk premium)이 연 약 3%라는 의미.

## GRS 검정: 모형의 전체적 적합도

### 왜 필요한가

글로벌 3팩터 모형으로 48개 포트폴리오를 회귀하면, 각각 알파(α)가 나온다. 이 48개 알파가 **전부 동시에 0인가?**를 테스트한다.

개별적으로는 유의하지 않더라도, 48개가 체계적으로 한 방향이면 모형이 잘못된 것이다.

### Gibbons, Ross, Shanken (1989) 검정

$$\text{GRS F-stat} = \frac{T-N-K}{N} \times \frac{\hat{\alpha}'\hat{\Sigma}^{-1}\hat{\alpha}}{1 + \hat{\mu}_f'\hat{\Omega}_f^{-1}\hat{\mu}_f}$$

(수식이 복잡하지만 핵심만: "모든 알파가 동시에 0이라는 가설"을 검정하는 F-통계량)

### 해석

| GRS F-stat | p-value | 의미 |
|-----------|---------|------|
| 낮음 (< 2) | > 0.05 | 모형이 기각되지 않음 — 적합 |
| 높음 (> 3) | < 0.01 | **모형이 기각됨** — 불완전 |

### Table VI 읽는 법

```
GRS F-Stat: 2.66    p-value: 0.000
```
→ 글로벌 3팩터 모형도 GRS 검정에서는 기각된다 (p < 0.001). 하지만 다른 모형들(CAPM: F=6.02, FF 6팩터: F=7.11)보다 훨씬 낫다.

> **핵심**: GRS에서 기각되었다 ≠ 쓸모없다. 모든 모형이 기각되는 상황에서, **상대적으로 가장 덜 기각되는** 모형이 최선이다. 글로벌 3팩터가 그 자리를 차지한다.

## Eig% (공분산 설명력): 모형이 포착하는 공동 움직임

Table VI의 마지막 열: "% of Covariances"

이것은 **모형이 함축하는 공분산 행렬**이 **실제 공분산 행렬**의 몇 %를 설명하는가를 나타낸다.

$$\text{Eig\%} = \frac{\sum \text{모형 함축 고유값}}{\sum \text{표본 고유값}}$$

| 모형 | Eig% |
|------|------|
| CAPM | 57% |
| CAPM + 유동성 | 64% |
| 글로벌 3팩터 | **84%** |

→ 글로벌 3팩터 모형이 48개 포트폴리오 수익률의 공동 움직임 중 84%를 포착한다.

## 요약: 테이블을 읽을 때 체크리스트

| 숫자 | 확인사항 |
|------|---------|
| Mean | 연환산? 월간? 양수? |
| (t-stat) | ≥ 2면 유의, < 1이면 무시 |
| Stdev | 변동성이 어느 정도인가 |
| Sharpe | 0.5 이상이면 괜찮음, 1 이상이면 좋음 |
| Alpha | 시장 보정 후에도 수익이 남는가 |
| Correlation | 양이면 같이, 음이면 반대, 크기 = 강도 |
| $R^2$ | 설명력 %, 0.5 이상이면 상당히 잘 설명 |
| GRS F | 작을수록 모형이 나음, p > 0.05면 미기각 |

## 이 챕터에서 배운 것

이 도구들로 무장하면, 이 논문의 어떤 테이블이든 읽을 수 있다:
- **Mean/Stdev/Sharpe**: 전략의 수익, 위험, 효율
- **t-stat**: 진짜인가 우연인가
- **Correlation**: 전략 간 관계
- **Alpha/Beta/$R^2$**: 모형으로 설명 후 남는 것
- **PCA**: 공통 요인 추출
- **Fama-MacBeth**: 횡단면 가격결정
- **GRS**: 모형의 전체 적합도
