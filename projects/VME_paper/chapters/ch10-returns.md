> Section II.A (p.939~947)를 해체한다. Table I은 이 논문에서 가장 큰 테이블이다 — 5페이지, 3개 패널, 수백 개의 숫자. 이 챕터를 읽고 나면 Table I의 모든 숫자를 읽을 수 있다.

## Table I의 구조: 먼저 지도를 보자

Table I은 3개 패널로 구성된다:

| 패널 | 내용 | 자산군 |
|------|------|--------|
| **Panel A** | 개별주식 포트폴리오 | 미국, 영국, 유럽, 일본 + 글로벌 주식 |
| **Panel B** | 비주식 자산 포트폴리오 | 국가지수, 통화, 채권, 원자재 + 글로벌 비주식 + 글로벌 전체 |
| **Panel C** | 채권 대안 가치 측정치 | 5년 금리변화, 실질금리, 텀스프레드, 복합 |

각 행(row)에는 6개 통계량:
- **Mean**: 연환산 평균 초과수익률
- **(t-stat)**: 평균의 t-통계량
- **Stdev**: 연환산 표준편차
- **Sharpe**: 샤프 비율
- **Alpha**: 시장 대비 초과수익 (CAPM 알파)
- **(t-stat)**: 알파의 t-통계량

각 열(column) 블록:
- **가치 포트폴리오**: P1(저), P2(중), P3(고), P3-P1(스프레드), Factor(시그널 가중)
- **모멘텀 포트폴리오**: P1(저), P2(중), P3(고), P3-P1, Factor
- **50/50 조합**: P3-P1, Factor
- **마지막 행**: Correlation(Val, Mom) — 가치-모멘텀 상관계수

### 읽는 법

```
         Value Portfolios          Momentum Portfolios      50/50
         P1    P2    P3   P3-P1  Factor   P1   P2   P3  P3-P1 Factor  P3-P1 Factor
Mean    9.5%  10.6% 13.2% 3.7%   3.9%   8.8%  9.7% 14.2% 5.4%  7.7%   4.6%  5.8%
(t-stat)(3.31)(4.33)(5.19)(1.83) (1.66) (2.96)(4.14)(4.82)(2.08)(2.84) (3.98)(5.40)
```

- P1 = 하위 1/3 (가치가 낮은/모멘텀이 낮은)
- P3 = 상위 1/3 (가치가 높은/모멘텀이 높은)
- **P3-P1** = 스프레드 = 롱숏 전략의 수익
- **Factor** = 시그널 가중 팩터의 수익

## Panel A: 개별주식 — 4개국 + 글로벌

### 미국 주식 (1972.01~2011.07)

#### 가치

| 포트폴리오 | Mean | t-stat | Stdev | Sharpe | Alpha | t-stat |
|-----------|------|--------|-------|--------|-------|--------|
| P1 (저 BE/ME = 비싼 주식) | 9.5% | 3.31 | 17.9% | 0.53 | | |
| P2 (중간) | 10.6% | 4.33 | 15.4% | 0.69 | | |
| P3 (고 BE/ME = 싼 주식) | 13.2% | 5.19 | 15.9% | 0.83 | | |
| **P3-P1 (스프레드)** | **3.7%** | **1.83** | **12.8%** | **0.29** | **5.3%** | **2.66** |
| Factor (시그널 가중) | 3.9% | 1.66 | 14.8% | 0.26 | 5.8% | 2.49 |

**읽는 법**:
- 싼 주식(P3)이 비싼 주식(P1)보다 연 3.7% 더 번다 (= 가치 프리미엄)
- t-stat 1.83 → 5% 유의수준에 아슬아슬 (유의하지 않지만 10% 수준에서는 유의)
- 하지만 **알파** = 5.3%, t = 2.66 → 시장 위험 보정 후에는 **매우 유의**
- 왜 차이? CAPM 베타가 가치주에서 높기 때문. 시장 노출을 제거하면 순수 가치 효과가 더 강해짐

#### 모멘텀

| 포트폴리오 | Mean | t-stat | Stdev | Sharpe |
|-----------|------|--------|-------|--------|
| P1 (과거 수익 낮음 = 루저) | 8.8% | 2.96 | 18.6% | 0.47 |
| P3 (과거 수익 높음 = 위너) | 14.2% | 4.82 | 18.5% | 0.77 |
| **P3-P1** | **5.4%** | **2.08** | **16.4%** | **0.33** |
| Factor | 7.7% | 2.84 | 17.0% | 0.45 |

- 위너(P3)가 루저(P1)보다 연 5.4% 더 번다 (= 모멘텀 프리미엄)
- t = 2.08 → **유의** (5% 수준)
- 시그널 가중 팩터: 7.7%, t = 2.84 → 더 강함

#### 50/50 조합

| | P3-P1 | Factor |
|--|-------|--------|
| Mean | 4.6% | 5.8% |
| t-stat | 3.98 | 5.40 |
| Stdev | 7.2% | 6.8% |
| Sharpe | 0.63 | **0.86** |

**핵심 관찰**:
- 가치만: 샤프 0.26~0.29
- 모멘텀만: 샤프 0.33~0.45
- **50/50 조합**: 샤프 **0.63~0.86** — 개별보다 훨씬 높다!
- t-stat **5.40** — 극도로 유의

왜 이렇게 좋아지는가? **음의 상관** 때문이다.

#### 가치-모멘텀 상관

> Correlation (Val, Mom) = **-0.65**

가치와 모멘텀의 월간 수익률 상관이 -0.65. 매우 강한 음의 상관. 하나가 올라갈 때 다른 하나는 내려간다 → 결합하면 변동성이 급격히 줄어든다.

### 일본 주식 (1974.01~2011.07)

| 전략 | P3-P1 Mean | t-stat | Sharpe | Factor Sharpe |
|------|-----------|--------|--------|--------------|
| 가치 | **12.0%** | **4.31** | **0.79** | 0.77 |
| 모멘텀 | 1.7% | 0.54 | 0.09 | 0.13 |
| **50/50** | **6.4%** | **4.28** | **0.78** | **0.88** |
| 상관(Val,Mom) | | **-0.64** | | |

**일본의 특이점**:
- 가치: **12.0%**, t = 4.31 — 4개국 중 압도적 1위
- 모멘텀: **1.7%**, t = 0.54 — **통계적으로 유의하지 않음!**

일본은 모멘텀이 작동하지 않는 유일한 시장이다. 하지만:
1. 가치가 엄청나게 강하다
2. 가치-모멘텀 상관이 여전히 -0.64로 강하다
3. 50/50 조합은 여전히 잘 작동한다 (샤프 0.78~0.88)

> 논문 p.945: "rather than explain why momentum did not work in Japan, it would be nearly equally appropriate to ask why value did so well"

모멘텀의 "실패"를 설명하는 대신, 가치의 "성공"을 설명하는 것이 동등하게 적절하다. 둘은 동전의 앞뒷면이다.

### 글로벌 주식 (4개국 결합)

4개국을 **변동성의 역수로 가중**하여 결합한다 (각 시장이 동일한 위험을 기여하도록).

| 전략 | P3-P1 Mean | t-stat | Sharpe | Factor Sharpe |
|------|-----------|--------|--------|--------------|
| 가치 | 6.3% | 6.52 | 1.04 | 0.51 |
| 모멘텀 | 5.6% | 2.94 | 0.47 | 0.59 |
| **50/50** | **6.3%** | **7.09** | **1.28** | **0.60** |
| 상관(Val,Mom) | | **-0.60** | | |

> 각주 11 (p.945): 변동성 가중 방법 — 각 시장의 벤치마크 포트폴리오(수동 시장 수익)의 월간 표준편차의 역수를 가중치로 사용. 모든 포트폴리오(Low, Mid, High, 팩터)에 동일한 가중치 적용.

**핵심**: 글로벌 가치의 샤프 비율(1.04)이 개별 시장 평균보다 높다 ≠ 더 좋다. 이것은 **글로벌 가치 전략들이 서로 상관되어 있다**는 증거. 단순 분산이라면 평균보다 높아지지 않기 때문.

## Panel C: 채권 대안 가치 측정치

Panel B에서 채권의 가치 효과가 약했다. 5년 금리변화라는 단순 측정치의 한계일 수 있다. Panel C는 더 정교한 측정치를 시도한다:

| 가치 측정치 | 의미 | Sharpe | Alpha t-stat |
|-----------|------|--------|-------------|
| 5년 금리변화 (원래) | 금리가 많이 올라서 싸진 것 | 0.18 | 1.68 |
| 실질금리 (10년 금리 - 5년 인플레이션 전망) | 실질 수익이 높은 것 | 0.36 | 2.02 |
| 텀스프레드 (10년 금리 - 단기 금리) | 장기 프리미엄이 큰 것 | 0.25 | 3.03 |
| **복합 (세 가지 평균)** | 다양한 정보 결합 | **0.91~1.10** | **4.40~5.48** |

**핵심 발견**: 여러 가치 측정치를 **결합**하면 샤프 비율이 0.91~1.10으로 폭등한다. t-통계량도 4.40~5.48으로 매우 유의.

> "we are able to produce even more reliable risk premia when using multiple measures of value simultaneously that diversify away measurement error and noise across the variables" (p.947)

**시사점**: 실전 구현에서는 단일 측정치보다 여러 측정치의 복합이 훨씬 강력하다. 이 논문이 단일 측정치를 고수하는 것은 **학술적 보수성** 때문이지, 최적 전략이어서가 아니다.

## 전체 패턴 요약

### 패턴 1: 가치와 모멘텀은 어디서나 양(+)의 수익

8개 자산군 중 **모든 곳**에서 가치와 모멘텀 스프레드가 양수다. 통계적 유의성은 자산군마다 다르지만, 방향은 일관적이다.

### 패턴 2: 가치와 모멘텀은 어디서나 음(-)의 상관

| 자산군 | 상관계수 |
|--------|---------|
| 미국 주식 | -0.65 |
| 영국 주식 | -0.64 |
| 유럽 주식 | -0.55 |
| 일본 주식 | -0.64 |
| 국가 지수 | -0.43 |
| 통화 | -0.42 |
| 채권 | -0.53 |
| 원자재 | -0.60 |
| **평균** | **약 -0.56** |

**전부 음수**. 예외 없음.

### 패턴 3: 조합이 개별보다 항상 더 낫다

모든 자산군에서 50/50 조합의 샤프 비율이 가치 또는 모멘텀 단독보다 높다. 음의 상관 + 양의 수익 = 분산 효과.

### 패턴 4: 시그널 가중이 P3-P1보다 약간 더 낫다

거의 모든 경우에서 Factor(시그널 가중)의 샤프 비율이 P3-P1 스프레드보다 높다. 왜? 시그널의 연속적 정보를 더 잘 활용하고, 더 분산된 포트폴리오이기 때문.
