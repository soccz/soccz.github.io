Overview

분석의 구조

손흥민 선수의 경기 데이터를 크롤링하고, 전처리 과정을 거쳐 승리 확률을 예측하는 머신러닝 모델을 구축합니다. PCA와 랜덤 포레스트를 통한 변수 선택, SVM 모델 학습까지의 과정을 담고 있습니다.

데이터 수집
웹 크롤링을 통해 손흥민 선수의 경기별 세부 데이터를 수집합니다. JSON 형식으로 저장된 원시 데이터를 통합하고 정제합니다.
전처리 및 변수 선택
결측값 처리, 라벨링, 상관관계(corr) 분석으로 숫자 데이터로 변환합니다. 랜덤 포레스트로 결과와 연관성이 높은 핵심 변수를 추출합니다.
예측 모델
선택된 변수를 기반으로 SVM 모델을 학습하여 경기 승리 확률을 예측합니다. PCA 시도(실패)를 포함한 모델 선택 과정을 기록합니다.
