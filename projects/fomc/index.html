Overview

프로젝트 개요

FOMC(연방공개시장위원회) 회의 전후의 뉴스 데이터를 다양한 NLP 기법으로 분류하고 키워드를 추출하여 통화정책 방향성의 사전 신호를 포착하는 연구입니다.

뉴스 데이터 수집
종목 리스트 기반으로 FOMC 관련 뉴스 데이터를 수집하고 전처리합니다. end2.py를 통해 CSV 형태의 정제된 데이터셋을 생성합니다.
RAG 기반 분류
Retrieval-Augmented Generation을 활용하여 FOMC 관련 뉴스를 정밀하게 분류합니다. 가장 완성도 높은 형태의 분석 파이프라인입니다.
키워드 추출
FOMC 발표 30일 전의 뉴스 데이터를 기반으로 핵심 키워드를 추출하여 통화정책 변화의 선행 지표를 탐색합니다.
Approaches

5가지 분석 접근법

RAG 모델의 성능과 컴퓨팅 자원의 한계를 고려하여 다양한 접근법을 실험했습니다.

RAG_fomc_keyword
가장 완성된 형태. RAG를 통해 FOMC 관련 뉴스 데이터를 정밀하게 분류하는 파이프라인
noRAG v1
RAG 모델의 컴퓨팅 한계로 인해 단순 키워드 기반 분류 방식으로 전환한 접근법
noRAG v2
가장 단순한 형태이나 의미 있는 결과를 보여주는 키워드 매칭 방식
noRAG v3
FOMC 발표 30일 전의 뉴스 데이터를 활용하여 사전 키워드를 추출하는 시간 윈도우 기반 접근
RAG v1/v2
컴퓨팅 한계 극복을 위해 모델 호출 최소화(v1)와 병렬 처리(v2)를 시도했으나 한계에 직면
