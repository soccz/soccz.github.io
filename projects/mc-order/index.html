Overview

프로젝트 소개

MC_order는 소규모 매장 운영자를 위한 주문 관리 웹앱입니다. 주문 생성부터 완료까지의 전 과정을 추적하고, 메뉴와 재고를 효율적으로 관리할 수 있도록 설계했습니다.

주문 접수
고객 주문을 실시간으로 접수하고 대기열에 등록합니다. 메뉴 항목별 수량, 옵션, 특이사항을 기록합니다.
상태 추적
접수 → 준비 중 → 완료 → 수령의 단계별 상태를 실시간으로 전환하고 추적합니다.
메뉴 관리
메뉴 항목 추가/수정/삭제, 가격 설정, 카테고리 분류, 품절 처리를 관리자 화면에서 수행합니다.

Architecture

시스템 구조

프론트엔드와 백엔드가 분리된 구조로, REST API를 통해 데이터를 주고받습니다.

Frontend
SPA 구조의 클라이언트 애플리케이션. 반응형 레이아웃으로 모바일과 태블릿 환경을 지원합니다.
Backend API
RESTful API 서버. 주문 CRUD, 메뉴 관리, 사용자 인증 엔드포인트를 제공합니다.
Database
주문, 메뉴, 사용자 데이터를 영구 저장합니다. 주문 이력은 날짜별로 인덱싱하여 빠른 조회를 지원합니다.
