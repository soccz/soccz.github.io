왜 이 강의인가?

대부분의 파이썬 강의는 문법을 암기하는 데 집중합니다. for 루프를 어떻게 쓰는지는 알려주지만, 파이썬이 내부적으로 iter()와 next()를 호출한다는 사실은 말하지 않습니다.

이 강의는 다릅니다. 파이썬 인터프리터가 실제로 어떻게 생각하는지를 보여줍니다. AI가 코드를 생성하는 시대에, 그 코드를 읽고 판단할 수 있는 능력이 진짜 실력입니다.

Python 3.14 공식 문서를 기반으로, 초등학생도 이해할 수 있는 비유부터 CPython 바이트코드 수준의 내부 구현까지 — 하나의 커리큘럼에 담았습니다.

  • 🔍
    De-sugaring 방식 for문을 iter()/next()/StopIteration으로 분해해서 이해합니다
  • 📖
    공식 문서 기반 Python 공식 튜토리얼의 개념 흐름과 용어를 그대로 사용합니다
  • 🤖
    AI 시대 관점 AI가 생성한 코드를 읽고, 검증하고, 판단하는 능력을 키웁니다
  • ⚠️
    실수 포인트 명시 Mutable 기본값, 클로저 캡처 등 실제로 자주 틀리는 함정을 짚어줍니다
  • 🧪
    직접 실행 중심 모든 개념마다 "지금 바로 해봐" 블록으로 즉시 확인합니다
